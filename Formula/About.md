# Anki-Cards/Formula
A flashcard for a formula.

## Fields

|Name|Represents|
|:--|:--|
|Name|The name of the formula.|
|Formula|The formula, preferably using LaTeX.|
|Explanation|The explanation of the formula. (Optional)|
|Proof|The proof of the formula. (Optional)|
